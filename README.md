# Universal Pine

静的HTMLで構築したUniversal Pine社のコーポレートサイトです。

**本番URL**: [https://universalpine.com]

## プロジェクト概要

- AIテクノロジーに特化した企業のコーポレートサイト
- 静的HTMLとCSSで構築
- Vercelでホスティング

## 主要ページ

- ホームページ (index.html)
- 会社概要 (about.html)
- 製品情報 (product.html)
- ニュース (news.html)
- お問い合わせ (contact.html)
- 採用情報 (recruit.html)

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## デプロイ方法

```bash
# ビルド
npm run build

# デプロイ
npm run deploy
```
